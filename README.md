# 십이장기

### 말을 움직일 경우 
(예: 王을 B2로 움직인다 했을때 -> Kb2 )

### 말을 잡을 경우 @로 표시한다.

### 말을 사용하지 않는 경우 x로 표시한다.

### 잡은 포로를 놓는 경우 !를 뒤에 붙인다.

---
### **말**

|말|왕|상|장|자|후|
|--|--|--|--|--|--|
|**영문**|K|S|J|Z|H|
|**번호**|0|1|2|3|4|
---
### **좌표**

|Y\X|1|2|3|4|
|-|-|-|-|-|
|**a**|a1|a2|a3|a4|
|**b**|b1|b2|b3|b4|
|**c**|c1|c2|c3|c4|
---
### **상대적 좌표 번호**

|Y\X|1|0|-1|
|-|-|-|-|
|**1**|1|2|3|
|**0**|4|5|6|
|**-1**|7|8|9|
